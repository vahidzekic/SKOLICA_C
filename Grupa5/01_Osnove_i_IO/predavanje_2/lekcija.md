# Predavanje 2: Standardni Ulaz i Format Specifikatori

## Teorijsko Objašnjenje
Funkcija `scanf` se koristi za čitanje podataka sa standardnog ulaza (tastature). Zahteva format specifikatore koji definišu tip podatka koji se čita, kao i adrese promenljivih (`&`).

## Format Specifikatori
- `%d` - ceo broj (`int`)
- `%f` - realan broj (`float`)
- `%c` - jedan karakter (`char`)
- `%s` - string (niz karaktera)
